## Hosting providers

### GitHub Pages
- Provider: GitHub Inc. (subsidiary of Microsoft Corporation)
- Location: USA (primary), CDN globally
- Data: IP address, browser info, access time in server logs
- Privacy: https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement
- Transfer basis (DSGVO): Standard Contractual Clauses (SCC)

### Netlify
- Provider: Netlify Inc., USA
- Data: IP, browser, access logs
- Privacy: https://www.netlify.com/privacy/

### Vercel
- Provider: Vercel Inc., USA
- Data: IP, browser, access logs
- Privacy: https://vercel.com/legal/privacy-policy

### Hostpoint (CH)
- Provider: Hostpoint AG, Rapperswil-Jona, Switzerland
- Data: IP, access logs, stored in Switzerland
- Privacy: https://www.hostpoint.ch/hostpoint/datenschutz.html

## Web fonts

### Google Fonts (via <link> tag)
- Provider: Google LLC, USA
- Data transferred: IP address of visitor on every page load
- When: at page load, before user interaction
- Mitigation: self-host fonts to eliminate this transfer entirely
- Self-hosting tool: https://google-webfonts-helper.herokuapp.com

### Google Fonts (self-hosted)
- Fonts downloaded once by operator, served from own domain
- No data transferred to Google on visitor page load
- Recommended approach for maximum privacy

### Bunny Fonts
- Provider: BunnyWay d.o.o., Slovenia (EU)
- Privacy-friendly alternative to Google Fonts
- GDPR/nDSG compliant, no personal data logged
- URL: https://fonts.bunny.net

## Analytics

### Plausible Analytics
- Provider: Plausible Insights OÜ, Estonia (EU)
- No cookies, no personal data, no cross-site tracking
- Data: page views, referrers, device types (aggregated)
- Privacy: https://plausible.io/privacy

### Fathom Analytics
- Provider: Conva Ventures Inc., Canada
- No cookies, GDPR/nDSG compliant
- Privacy: https://usefathom.com/privacy

### Matomo (self-hosted)
- All data stays on your own server
- Can be configured without cookies
- Most privacy-friendly analytics option
